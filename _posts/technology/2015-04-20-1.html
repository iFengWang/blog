VVDocumenter-Xcode（生成注释）：

1、git clone http://github.com/onevcat/VVDocumenter-Xcode
2、打开工程build&run一下就安装成功了
3、重启xcode
4、使用:在要加入注释的地方输入///即可自动转换成指定格式

appledoc（将注释生成文档）：

安装:

git clone git://github.com/tomaz/appledoc.git
cd appledoc
sudo sh install-appledoc.sh

查看:

appledoc --help
appledoc --version

使用


appledoc --no-create-docset --project-name "项目" --project-company "公司” --company-id "com.公司.项目" --output 输出路径 源路径
--no-create-docset:单独生成HTML且第一次不要加。
--keep-intermediate-files:第一次生成时包含很多中间文件，包括HTML。
选中.h文件中要加注释的接口，右击或alt+command+/,选择“生成注释文档”。
